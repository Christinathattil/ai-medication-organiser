# Senior-Friendly Sound Files

This directory contains audio notification sounds for the medication manager.

## Required Sound Files

You need to add these MP3 files to this directory:

### 1. reminder.mp3
- **Purpose**: Main medication reminder sound
- **Recommended**: Gentle bell or chime
- **Duration**: 2-3 seconds
- **Free source**: https://freesound.org/ or https://www.zapsplat.com/

### 2. success.mp3
- **Purpose**: Success feedback (medication marked as taken)
- **Recommended**: Pleasant "ding" or positive tone
- **Duration**: 1-2 seconds

### 3. alert.mp3
- **Purpose**: Warnings or important alerts
- **Recommended**: Attention-grabbing but not alarming
- **Duration**: 2 seconds

## How to Get Free Sounds

### Option 1: Freesound.org
1. Go to https://freesound.org/
2. Search for "gentle bell", "notification", or "chime"
3. Filter by "CC0" license (public domain, no attribution required)
4. Download and rename to match the filenames above

### Option 2: Zapsplat.com
1. Go to https://www.zapsplat.com/
2. Free account required (no payment)
3. Search for notification sounds
4. Download and rename

### Option 3: Create Simple Tones
You can use online tools like:
- https://www.beepbox.co/ - Create custom tones
- https://onlinetonegenerator.com/ - Generate simple beeps

## Fallback

If you don't add sound files, the app will still work but silently fail when trying to play sounds. No error will be shown to users.

## File Format

- **Format**: MP3 (best browser support)
- **Bitrate**: 128kbps is sufficient
- **Size**: Keep under 100KB each for fast loading
